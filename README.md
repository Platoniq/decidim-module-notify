# Decidim::Notify

A note-taker feature focused on conversations.

## Usage

Notify will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'decidim-notify
```

And then execute:

```bash
bundle
```
